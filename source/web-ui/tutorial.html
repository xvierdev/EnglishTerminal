Tutorial de como usar o EnglishTerminal

Antes de executar

1. Caso esteja usando o arquivo .exe o EnglishTerminal.exe não precisa fazer nada antes de executar; 

2. Agora se você estiver usando o código-fonte aí você vai ter que executar o esses comandos(certifique-se que a o python está instalado se não vai dar erro e você deve estar na pasta do repositório):

    #Cria um Ambiente Virtual
    python -m venv EnglishTerminal-venv
    #Ativa o Ambiente Virtual
    source EnglishTerminal-venv/bin/activate

    #Atualiza o pip
    pip install --upgrade pip
    #Instala as depedências
    pip install -r requeriments.txt
    #Executa o programa
    python source/main.py

E por motivos de simplicidade do código eu não irei falar como se usa porque esse cara é muito simples só está em inglês

Mas sempre quando executar ele não se esqueça de executar esses comandos dentro da pasta do repositório:

    #Ativa o Ambiente Virtual
    source EnglishTerminal-venv/bin/activate
    
    #Executa o programa
    python source/main.py

mas se você usar o executável com o wine PODE dar certo, mas poderá aver alguns erros, ah e lembre-se que ele detecta o sistema operacional da máquina que está sendo executado
